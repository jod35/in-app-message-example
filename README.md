## Run Android
```
git clone https://github.com/KirillGudkov/in-app-message-example
cd <proj_dir>/in-app-message-example
yarn install
react-native run-android
```

## Run iOS
```
git clone https://github.com/KirillGudkov/in-app-message-example
cd <proj_dir>/in-app-message-example
yarn install
cd ios
pod install
cd ..
react-native run-ios // or use Xcode
```
